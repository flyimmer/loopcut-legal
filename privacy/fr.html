Politique de Confidentialité

Dernière mise à jour : 9 avril 2026
LoopCut est une application de bien-être numérique qui vous aide à briser vos habitudes d'utilisation du téléphone. Nous nous engageons à protéger votre vie privée tout en vous offrant des fonctionnalités qui vous aident à développer des habitudes téléphoniques plus saines.

Les Données Que Nous Collectons

Informations sur le Compte

Lorsque vous créez un compte LoopCut (via la connexion Google), nous collectons :

  • Adresse e-mail du compte Google — pour la connexion et la récupération du compte
  • Nom d'affichage (facultatif) — pour la personnalisation
  • Statut du compte — si votre compte a été activé (utilisateur gratuit vs enregistré)

Analyses d'Utilisation

Pour améliorer l'expérience de l'application, LoopCut peut collecter des données d'utilisation agrégées et anonymisées, telles que :

  • Les fonctionnalités les plus fréquemment utilisées (ex. : mode heure du coucher, points de contrôle)
  • Le nombre de sessions et les modèles d'utilisation généraux
  • Les rapports de plantage et d'erreur de l'application

Ces données sont anonymisées — elles ne peuvent pas être utilisées pour vous identifier personnellement. Elles sont utilisées uniquement pour comprendre comment l'application est utilisée et pour l'améliorer.

Vous pouvez désactiver les analyses d'utilisation à tout moment dans les paramètres de l'application.

Données NON Collectées

  • ❌ Aucun contenu d'autres applications (ce que vous lisez, tapez ou visionnez)
  • ❌ Aucune donnée de localisation
  • ❌ Aucun contact ou journal d'appels
  • ❌ Aucun identifiant publicitaire ou suivi des annonces
  • ❌ Aucune donnée vendue à des tiers

Synchronisation Cloud

Si vous vous connectez avec un compte Google, les données suivantes peuvent être synchronisées avec nos serveurs :

  • Vos paramètres et préférences de l'application
  • Historique et statistiques des interventions
  • Statut du compte

La synchronisation Cloud est conçue pour vous permettre de restaurer vos données sur un nouvel appareil. Vos données sont stockées de manière sécurisée à l'aide de Firebase (Google Cloud) et ne sont pas partagées avec des tiers.

Vous pouvez utiliser LoopCut sans compte — toutes les fonctionnalités fonctionnent entièrement hors ligne avec un stockage uniquement local.

Service d'Accessibilité — Déclaration Détaillée

LoopCut utilise l'AccessibilityService d'Android exclusivement pour détecter quelle application est actuellement au premier plan. Le service :

  • Écoute uniquement les événements TYPE_WINDOW_STATE_CHANGED (changements d'application)
  • Lit uniquement le nom du package de la fenêtre active (ex. : com.android.chrome)
  • Ne lit jamais le contenu de l'écran, le texte, les étiquettes de l'interface utilisateur ou les saisies de l'utilisateur
  • N'enregistre ni ne transmet jamais ce que vous faites à l'intérieur d'une application
  • Peut être désactivé à tout moment dans Paramètres Android → Accessibilité

Conservation et Suppression des Données

  • Les données locales sont supprimées lorsque vous désinstallez LoopCut
  • Les données du compte peuvent être supprimées en allant dans Paramètres → Compte → Supprimer le compte dans l'application, ou en nous envoyant un e-mail
  • Les données d'analyse sont anonymisées et ne peuvent pas être retracées jusqu'aux utilisateurs individuels
  • Vous pouvez demander une copie ou la suppression des données de votre compte Firebase en nous contactant

Confidentialité des Enfants

LoopCut ne collecte pas sciemment d'informations personnelles auprès d'enfants de moins de 13 ans. Si vous pensez qu'un enfant nous a fourni des données personnelles, veuillez nous contacter afin que nous puissions les supprimer.

Modifications de Cette Politique

Si nous mettons à jour cette politique de confidentialité, la nouvelle version sera publiée à cette URL avec une date de mise à jour. Nous informerons les utilisateurs des modifications importantes via l'application.
